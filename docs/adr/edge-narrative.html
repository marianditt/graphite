Edge Narrative

Edges are narrative: every declared edge must be referenced inline in the body using a special marker that names the target node ID. Edges are never auto-rendered as standalone sections. The linter enforces that every declared target appears as a marker in the body, and every marker in the body has a matching declaration. The renderer replaces these markers with hyperlinks to the target node's page.

This keeps the graph connected to prose — edges are woven into the explanation, not separated into metadata tables.
