Evidence anchors use @graphite:evidence <id> in line-comment languages (//, #, <!-- -->). For comment-less formats (JSON, data files), a .graphite sidecar file sits alongside the source and uses JSONPath or regex patterns.

Sidecar naming: source.ext → source.ext.graphite. Patterns never use line numbers — they resolve by content. See compiler for implementation details.
