Compiler

A Rust CLI that implements the Graphite compiler pipeline: parse, validate, resolve, render. Operates offline with no database, no state, and no plugin system.

The pipeline architecture is at ADR-003, the type system at ADR-008, evidence anchoring at ADR-002, HTML generation at ADR-007, and the tutoring error format at ADR-010.
